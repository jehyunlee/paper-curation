Public Profile Matters: A Scalable Integrated Approach to Recommend Citations in the Wild

저자: Karan Goyal, Dikshant Kukreja, Vikram Goyal, Mukesh Mohania (IIIT Delhi) | 날짜: 2026-03-18 | arXiv: 2603.17361v1 [cs.IR]

핵심 요약

본 논문은 local citation recommendation (LCR) 문제를 해결하기 위해 두 단계 시스템을 제안한다. 첫 번째 단계인 Profiler는 논문의 "public profile" (인용 네트워크 내에서 해당 논문이 어떻게 인식되는지)을 활용하는 비학습(non-learnable) 검색 모듈이며, 두 번째 단계인 DAVINCI는 검색 단계의 confidence prior와 semantic 정보를 adaptive vector-gating 메커니즘으로 융합하는 reranker이다. 또한 기존 transductive 평가 방식의 한계를 지적하고, 시간 순서를 엄격히 준수하는 inductive evaluation setting을 도입하여 현실적 평가 프레임워크를 제시한다.

연구 배경 및 동기

과학 논문의 폭발적 증가로 인해 연구자가 관련 선행 연구를 찾아 인용하는 것이 점점 어려워지고 있다. 기존 LCR 시스템은 크게 세 가지 한계가 있었다:

  • SymTax (Goyal et al., 2024)의 3단계 아키텍처(prefetcher-enricher-reranker)는 높은 계산 비용, confirmation bias 전파, taxonomy metadata 의존이라는 문제를 가짐
  • CiteBART (Celik & Tekir, 2025)의 generative 접근은 존재하지 않는 인용을 hallucinate하며, 저자명 기반의 피상적 매칭에 의존
  • 기존 평가 프로토콜은 transductive setting에서 수행되어 test 문서가 corpus에 포함되는 비현실적 상황을 허용

방법론

데이터셋

5개 벤치마크 데이터셋 사용: ACL-200, FullTextPeerRead, RefSeer (350만 contexts), arXiv (300만 contexts), ArSyTa (800만 contexts). 각 데이터셋에 대해 transductive 및 inductive setting의 통계를 비교 제시.

Inductive Evaluation Setting

  • 평가 문서와 후보 corpus를 엄격히 분리 (Deval ∩ C = ∅)
  • 시간적 일관성 제약: 후보 문서는 반드시 쿼리 문서보다 먼저 출판된 것만 허용
  • 이를 통해 실제 연구 현장에서 "새 논문에 대한 인용 추천" 시나리오를 시뮬레이션

Profiler (1단계: Non-Learnable Retrieval)

  • 각 문서의 base vector (specter2_base 인코딩)에 inward ego network의 정보를 융합하여 "profiled representation" 생성
  • ˆvi = vi + (1/|Nin(Di)|) * Σ(α·vsji + β·vj), 여기서 α+β=1 (non-learnable hyperparameters)
  • citation이 없는 새 논문은 자동으로 base vector로 fallback (cold-start 대응)
  • 쿼리도 citation context와 meta information의 가중 결합으로 구성
  • Cosine similarity로 후보 검색 → confidence score도 함께 산출

DAVINCI (2단계: Reranking)

  • Prior Discriminator: raw retrieval score → ordinal abstraction (rank 변환) → exponential decay remapping (pi = λ^ri, λ=0.95)
  • Adaptive Gated Fusion: SciBERT로 query-candidate pair 인코딩 → Text Projection Tower와 Score Projection Tower로 공통 latent space 매핑 → Gating Network이 element-wise soft mask 생성 → Hadamard product로 융합
  • Learning Objective: margin-based triplet loss로 직접 ranking 최적화
  • 총 110M 파라미터의 경량 모델

주요 결과

  • Profiler의 우수성: 모든 데이터셋에서 기존 Prefetcher+Enricher 조합 대비 MRR 45-64% 향상, 계산 시간 32-44배 단축 (예: ArSyTa에서 236시간 → 7.26시간)
  • End-to-End SOTA 달성: 5개 데이터셋 전체에서 BM25, SciNCL, HAtten, SymTax를 모든 metric(MRR, Recall@K, NDCG@K)에서 상회
  • 대규모 reranker 대비 우위: 110M 파라미터 DAVINCI가 Qwen3-Reranker-8B (8.19B)와 bge-reranker-v2-minicpm (2.72B)를 모든 데이터셋에서 능가 (최대 70배 작은 모델이 더 우수)
  • Ablation 결과: (1) confidence prior 제거 시 성능 하락, (2) discriminator 비활성화 시 하락, (3) softmax 대체 시 하락, (4) scalar gating 대체 시 가장 큰 하락 → vector-gating의 기여가 가장 큼
  • Candidate pool size 분석: k=300이 성능과 계산 비용의 최적 균형점

독창성 및 기여

  1. Public profile 개념의 도입: 논문의 "내재적 콘텐츠"뿐 아니라 연구 생태계 내에서의 "인식된 정체성"을 활용한다는 착안이 참신함. 인용 네트워크의 inward ego network을 non-learnable하게 활용하여 효율성과 bias 제거를 동시에 달성
  2. Inductive evaluation setting: 기존 LCR 연구의 transductive 평가가 현실을 반영하지 못한다는 근본적 문제를 지적하고, 시간적 제약을 적용한 엄격한 평가 체계를 제안. 이는 향후 LCR 연구의 표준이 될 가능성이 있음
  3. DAVINCI의 vector-gating 메커니즘: retrieval confidence와 semantic signal의 이질적 정보를 element-wise로 융합하는 설계가 scalar gating이나 단순 concatenation보다 효과적임을 실증
  4. 경량 모델의 domain-specific 설계 우위 입증: 110M 모델이 수십억 파라미터의 범용 모델을 능가하며, "architectural sophistication > raw scale" 논제를 강하게 뒷받침

한계 및 향후 연구

저자 언급 한계

  • 영어 및 컴퓨터 과학 분야에 한정된 평가 (인문/사회과학의 인용 패턴 차이 미검증)
  • "절대적 신규 논문"에 대한 cold-start 문제 (인용 네트워크에 아직 통합되지 않은 논문은 profiling 불가)
  • 512 토큰 제한으로 인한 긴 문서의 context 손실
  • 메타데이터 품질(누락된 초록, 잘못 파싱된 제목)에 대한 의존성

리뷰어 추가 지적

  • Reproducibility 우려: "acceptance 후 코드/모델 공개" 약속은 있으나 현재 검증 불가. 특히 inductive setting의 데이터 분할 기준이 재현 가능하려면 정확한 timestamp 처리 방법의 명시 필요
  • Profiler의 hyperparameter 일반화: α, β, γ, δ를 소규모 데이터셋에서 결정하고 대규모에 적용하는 전략이 domain 변경 시에도 유효한지 불확실
  • Popularity bias 완전 해소 미입증: 저자가 citation count, venue prestige 등 explicit signal을 제거했다고 주장하나, inward ego network 자체가 많이 인용된 논문에 더 풍부한 profile을 부여하므로 implicit popularity bias가 여전히 존재할 가능성이 있음
  • 단일 실행 결과 보고: 대규모 데이터셋에서의 계산 비용 문제는 이해하나, 최종 결과가 single run이라는 점은 통계적 신뢰도 측면에서 아쉬움. Subset에서의 3회 시행 안정성 분석만으로는 충분하지 않음
  • Inductive setting의 현실성: 실제 연구 현장에서는 preprint 단계에서 인용이 이루어지는 경우도 빈번한데, 공식 출판일 기준의 temporal constraint가 이를 반영하는지 불분명
  • Science of Science 관점에서의 분석 부재: 공공 프로필이 실제 과학적 영향력(scientific impact)과 어떻게 관련되는지, 이 시스템이 citation dynamics에 미치는 잠재적 영향에 대한 깊이 있는 논의가 부족

총평: Non-learnable retrieval과 lightweight reranking이라는 실용적 설계 철학 위에, inductive evaluation이라는 방법론적 기여까지 더해 LCR 분야에 의미 있는 진전을 이룬 논문이다. 특히 110M 모델이 수십억 파라미터 모델을 능가한다는 결과는 domain-specific 설계의 가치를 강하게 시사하지만, popularity bias의 암묵적 잔존과 단일 실행 보고는 보완이 필요하다.
