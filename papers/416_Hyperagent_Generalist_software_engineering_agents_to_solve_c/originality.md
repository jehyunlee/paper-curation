We present HYPERAGENT , a novel generalist multi-agent system that ad- dresses a broad spectrum of SE tasks across multiple programming languages by emulating the workflows of human developers.. HYPERAGENT consists of four specialized agents—Planner, Navigator, Code Editor, and Executor—capable of managing the full lifecycle of SE tasks, from initial planning to final verifica- tion.. HYPERAGENT achieves state-of-the-art results on diverse SE tasks, includ- ing GitHub issue resolution on the well-known SWE-Bench benchmark, surpass- ing strong baselines..
