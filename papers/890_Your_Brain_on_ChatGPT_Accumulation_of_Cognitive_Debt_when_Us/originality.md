Like any other tool, it carries its own set of advantages and limitations.. This study focuses on finding out the cognitive cost of using an LLM in the educational context of writing an essay.. We assigned participants to three groups: LLM group, Search Engine group, Brain-only group, where each participant used a designated tool (or no tool in the latter) to write an essay.. We conducted 3 sessions with the same group assignment for each participant.. In the 4th session we asked LLM group participants to use no tools (we refer to them as LLM-to-Brain), and the Brain-only group participants were asked to use LLM (Brain-to-LLM).. We recruited a total of 54 participants for Sessions 1, 2, 3, and 18 participants among them completed session 4..
