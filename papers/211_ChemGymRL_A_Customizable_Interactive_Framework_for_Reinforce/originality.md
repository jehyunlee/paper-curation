ChemGymRL: A customizable interactive framework for reinforcement learning for digital chemistry Chris Beeler,abc Sriram Ganapathi Subramanian,bd Kyle Sprague,b Mark Baula,b Nouha Chatti,b Amanuel Dawit,b Xinkai Li,b Nicholas Paquin,b Mitchell Shahen,b Zihan Yang,b Colin Bellinger,c Mark Crowley *b and Isaac Tamblynde This paper provides a simulated laboratory for making use of reinforcement learning (RL) for material design, synthesis, and discovery.. Since RL is fairly data intensive, training agents ‘on-the-fly’ by taking actions in the real world is infeasible and possibly dangerous.. Moreover, chemical processing and discovery involves challenges which are not commonly found in RL benchmarks and therefore offer a rich space to work in.. We introduce a set of highly customizable and open-source RL environments, ChemGymRL, implementing the standard gymnasium API..
