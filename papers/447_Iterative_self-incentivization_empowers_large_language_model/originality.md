To address these limitations, we propose EXSEARCH, an agentic search framework, where the LLM learns to retrieve useful information as the reasoning unfolds through a self-incentivized process.. At each step, the LLM decides what to retrieve (think- ing), triggers an external retriever (search), and extracts fine-grained evidence (recording) to support next-step reasoning.. To enable LLM with this capability, EXSEARCH adopts a Generalized Expectation-Maximization algorithm.. In the E-step, the LLM generates multiple search trajectories and assigns an importance weight to each; the M-step trains the LLM on them with a re-weighted loss function..
