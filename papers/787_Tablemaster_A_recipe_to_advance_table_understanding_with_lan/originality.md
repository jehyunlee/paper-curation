In this paper, we aim to enhance LMs for improved table understanding.. We identify four key challenges: 1) difficulty in locating target data, 2) deficiency in table semantics, 3) numerical inaccura- cies in textual reasoning, and 4) semantic inflexibility in symbolic reasoning.. To address these issues, we propose TableMaster, a recipe and comprehensive frame- work that integrates multiple solutions to overcome these obstacles.. TableMas- ter first extracts relevant table content and verbalizes it with enriched semantic context..
