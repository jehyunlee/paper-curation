The Adoption and Usage of AI Agents: Early Evidence from Perplexity

Figure 1

Essence

범용 AI 에이전트의 채택(adoption)과 사용(usage) 패턴을 대규모 현장 데이터로 실증 분석한 최초의 연구이다. Perplexity가 개발한 AI 브라우저 Comet과 그 내장 에이전트 Comet Assistant의 수억 건 익명화된 사용자 상호작용 데이터를 분석하여, "누가 AI 에이전트를 쓰는가?", "얼마나 집중적으로 쓰는가?", "무엇에 쓰는가?"라는 세 가지 근본적 질문에 답한다. 에이전트 사용 사례를 체계화하는 계층적 분류 체계(Agentic Taxonomy)를 새로 제안한 것이 핵심 기여이다.

Motivation (동기)

2025년은 "에이전틱 AI의 해"로 불리며, AI가 대화형 챗봇에서 행동 지향적 에이전트로 전환하고 있다. 그러나 실제 사용자들이 범용 AI 에이전트를 어떻게 채택하고 사용하는지에 대한 체계적 행동 증거는 거의 없었다. 기존 연구는 비대표적 기업 설문이나 코딩 에이전트 같은 특화 에이전트에 한정되었다. Comet이라는 실제 상용 제품의 대규모 데이터를 통해 이 공백을 메우고자 했다.

Achievement (성과)

  1. 채택 패턴 규명: 초기 채택자(pre-GA)가 전체 사용자의 30%이지만 에이전트 채택자의 50%, 에이전틱 쿼리의 70%를 차지. 첫 번째 코호트(7월 9일) 사용자는 GA 코호트 대비 채택 확률 2배, 쿼리량 9배
  2. 국가 수준 분석: 1인당 GDP(r=0.85)와 평균 교육연수(r=0.75)가 에이전트 채택과 강한 양의 상관관계
  3. 직업 클러스터 분석: Digital Technology(채택자 28%), 학생, 금융, 마케팅, 기업가 순으로 높은 채택률. 마케팅(AUR=1.46)과 기업가(AUR=1.38)가 채택 대비 사용 강도가 특히 높음
  4. 계층적 에이전틱 분류 체계: 7개 토픽 → 25개 서브토픽 → 90개 태스크의 3단계 분류 체계 개발. 생산성(36%)과 학습(21%)이 전체 쿼리의 57% 차지
  5. 사용 맥락 분석: 개인 55%, 직업 30%, 교육 16%. 시간 경과에 따라 인지 지향적 토픽(생산성, 학습, 커리어)으로 이동하는 경향

6. 환경(웹사이트) 집중도: docs.google.com(12%), 이메일 서비스(11%), linkedin.com(9%)이 상위 환경. 서브토픽별 집중도 편차가 큼(음악 97% vs 계정관리 28%)

How (방법론)

  • 3개 표본 설계: (A) 전체 Comet 사용자 모집단(수백만 명, 수억 건 쿼리)으로 채택·사용 강도 분석, (B) 10만 명 무작위 표본으로 O*NET 직업 클러스터 분류, (C) 10만 명 에이전트 사용자 표본으로 전체 에이전틱 쿼리의 사용 사례 분류
  • 분류 체계 개발: 상향식(bottom-up) 3단계 접근 -- 탐색(임베딩 + K-means 클러스터링) → 정제(수동 검토·병합·분할) → 분류(쿼리 분류 모델 적용). 1,000건 골든 데이터셋 대비 토픽 89%, 서브토픽 83%, 태스크 81% 정확도
  • 에이전틱 쿼리 정의: 브라우저 제어 또는 MCP/API를 통한 외부 앱 조작이 포함된 쿼리만 해당. 단순 웹 검색이나 코드 인터프리터는 제외(정보 교환 vs 환경 조작 구분)
  • 채택·사용 지표: Agent Adoption Ratio(AAR)와 Agent Usage Ratio(AUR)를 정의하여 세그먼트별 과대/과소 대표성 정량화

Originality (독창성)

  • 최초의 대규모 현장 연구: AI 에이전트(챗봇이 아닌)의 실제 사용 행동을 수억 건 데이터로 분석한 최초 사례
  • 에이전틱 vs Q&A 구분: 기존 LLM 사용 연구가 정보 교환 중심의 Q&A 쿼리를 다룬 것과 달리, 외부 환경을 실제로 조작하는 에이전틱 쿼리에 초점
  • 계층적 분류 체계의 범용성: Comet 특화가 아닌, 다른 에이전틱 제품에도 적용 가능하도록 설계된 분류 체계
  • 직업-사용사례 교차 분석: 직업 클러스터별 토픽 분포, 태스크 분포, 환경 분포를 체계적으로 매핑한 것은 기존 연구에 없던 시도

Limitation & Further Study (한계 및 향후 연구)

  • 초기 채택자 편향: Comet이 신제품이므로 표본이 기술 친화적 사용자에 치우쳐 있으며, 일반 인구 대표성은 부족
  • 짧은 관찰 기간: 2025년 7월 9일~10월 22일(약 3.5개월)로, 장기적 사용 패턴 변화를 체계적으로 분석하기에 불충분
  • 분류 노이즈: 에이전틱 쿼리 탐지, 분류 체계 레이블링, 직업 분류 모두 자동화된 분류기에 의존하므로 오분류(false positive/negative) 존재
  • 자동화 vs 증강 구분 불가: 사용자가 전체 태스크를 위임했는지, 일부만 위임했는지 구분할 수 없어 자동화/증강 비율 추정 불가
  • 단일 플랫폼 한계: Perplexity Comet 사용자만 대상이므로, OpenAI Operator나 Anthropic Computer Use 등 다른 에이전트 제품과의 비교 불가
  • 향후 연구: 모바일/크로스플랫폼 분석, 생산성 영향 측정, 에이전트 간 협업(A2A 프로토콜) 효과, 사용자 학습 곡선과 에이전트 활용 능력(agent literacy) 연구

Evaluation

Evaluation

총평: Perplexity Comet의 실제 사용 데이터로 AI 연구 에이전트 도입 패턴을 분석한 최초의 대규모 실증 연구로 현장 통찰이 풍부하다.
