Essence

In this study, we introduce ChemCrow, an LLM chemistry agent designed to accomplish tasks across organic synthesis, drug discovery, and materials design.

Motivation

  • Gap: Recently, large-language models (LLMs) have shown strong performance in tasks across domains, but struggle with chemistry-related problems.

Achievement
