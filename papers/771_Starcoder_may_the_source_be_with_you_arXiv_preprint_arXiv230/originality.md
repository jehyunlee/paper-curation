We fine-tuned StarCoderBase on 35B Python tokens, resulting in the creation of StarCoder.. We perform the most comprehensive evaluation of Code LLMs to date and show that StarCoderBase outperforms every open Code LLM that supports multiple programming languages and matches or outperforms the OpenAI code-cushman-001 model.. Furthermore, StarCoder outperforms every model that is fine-tuned on Python and still retains its performance on other programming languages.
