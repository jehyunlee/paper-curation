# Large language model agent for nuclear reactor operation assistance

> **저자**: Yoon Pyo Lee, Joowon Cha, Yonggyun Yu, Seung Geun Kim | **날짜**: 2025-12-01 | **DOI/URL**: https://doi.org/10.1016/j.net.2025.103842
> **리뷰 모드**: Web-only (Abstract 기반)

---

## Essence (본질)
외부 도구·문서·예측 모듈을 통합한 LLM 에이전트 아키텍처를 통해 원자력 발전소 시뮬레이터에서 이상 진단 및 운전 절차 자동 생성을 수행함으로써, 원자력 운전 보조 AI의 실현 가능성을 입증한다.

## Originality (독창성)
- **What**: 원자력 발전소 시뮬레이터 환경에서 LLM 에이전트 기반 운전 보조 시스템 개발
- **How**: 문서(RAG), 예측 모듈, 함수 호출을 통합한 에이전트 아키텍처를 구성하고, (1) 붕소 농도 이상 대응, (2) 신규 운전 절차서 자동 생성 두 실험으로 검증
- **Why**: 기존 딥러닝 기반 이상감지·자동화 연구를 확장하여, 자연어 추론과 외부 도구 활용이 가능한 에이전트 시스템이 원자력 분야에서도 포괄적 의사결정을 가능하게 함

## Summary (요약)
본 연구는 LLM 에이전트가 원자력 운전 시뮬레이터에서 붕소 농도 이상을 탐지·대응하고, 기존 매뉴얼을 기반으로 추가 학습 없이 새로운 운전 절차를 생성할 수 있음을 실험적으로 보여준다. 에이전트는 명령 실행과 운전 파라미터 조정이 가능해 안전하고 비용 효율적인 자동화된 원전 운전의 가능성을 제시한다. 원자력이라는 고위험 도메인에 LLM 에이전트를 적용했다는 점에서 AI4S의 도메인 응용 범위를 넓히는 의미 있는 연구이다.
