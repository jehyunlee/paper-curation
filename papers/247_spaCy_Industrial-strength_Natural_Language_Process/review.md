# spaCy: Industrial-strength Natural Language Processing in Python

> **저자**: Matthew Honnibal, Ines Montani | **날짜**: 2020 | **DOI/URL**: https://doi.org/10.5281/zenodo.1212303
> **리뷰 모드**: Web-only (Abstract 기반)

---

## Essence (본질)
산업 수준의 성능과 사용성을 갖춘 Python NLP 라이브러리 spaCy를 소개하며, 과학 텍스트 처리 및 정보 추출을 위한 실용적 NLP 인프라를 제공한다.

## Originality (독창성)
- **What**: 토크나이징, 품사 태깅, 개체명 인식, 의존구문 분석 등 핵심 NLP 파이프라인을 통합한 Python 라이브러리
- **How**: Cython 기반 고속 구현과 사전 훈련된 언어 모델을 결합해 연구·산업 양쪽에서 쉽게 사용 가능한 API 제공
- **Why**: 연구 목적 NLP 도구들은 대부분 성능·속도·사용성 중 하나를 희생하는데, spaCy는 세 가지를 동시에 달성하고자 함

## Summary (요약)
spaCy는 Explosion AI가 개발한 오픈소스 NLP 라이브러리로, 과학 논문 텍스트 파싱 및 정보 추출의 표준 도구로 널리 쓰인다. Zenodo를 통해 소프트웨어 논문으로 등록되어 있으며, ScispaCy 등 과학 도메인 특화 확장이 별도로 존재한다. AI4S 큐레이션 맥락에서는 과학 문헌 NLP 파이프라인의 기반 인프라로서 참조 문헌 역할을 한다.
