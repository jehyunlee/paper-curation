Motivated by the growing trend of scientific studies ac- companied by public code repositories, we propose TOOLMAKER, an agentic framework that autonomously transforms papers with code into LLM-compatible tools.. Given a GitHub URL and short task description, TOOLMAKER autonomously installs dependencies and gener- ates code to perform the task, using a closed- loop self-correction mechanism for debugging.. To evaluate our approach, we introduce a bench- mark comprising 15 complex computational tasks spanning various domains with over 100 unit tests to assess correctness and robust- ness.
