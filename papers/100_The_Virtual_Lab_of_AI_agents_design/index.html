The Virtual Lab of AI agents designs new SARS-CoV-2 nanobodies

저자: Kyle Swanson, Wesley Wu, Nash L. Bulaong, John E. Pak, James Zou | 날짜: 2025 | Journal: Nature | DOI: 10.1038/s41586-025-09442-9

Figure 1

Novelty: 5/5 Technical: 4/5 Significance: 5/5 Clarity: 4/5 Overall: 5/5

Essence

Virtual Lab은 LLM(GPT-4o) 기반 멀티 에이전트 시스템과 인간 연구자의 협업으로 학제간 과학 연구를 수행하는 프레임워크이다. PI 에이전트가 면역학자·계산생물학자·ML 전문가 에이전트를 이끌며, ESM + AlphaFold-Multimer + Rosetta를 조합한 나노바디 설계 파이프라인을 에이전트가 자체적으로 설계했다. 4개 시작 나노바디(Ty1, H11-D4, Nb21, VHH-72)에서 92개 돌연변이 나노바디를 생성하여 실험 검증한 결과, 93.5%가 양호한 발현량(>5 mg/L)을 보였고, H11-D4/Nb21 시리즈의 96%가 Wuhan RBD 결합력을 유지했으며, Nb21 돌연변이(I77V-L59E-Q87A-R37Q)가 JN.1 RBD에 결합(EC50 2.0 ng/mL)하고 KP.3 RBD에도 결합 증거를 보였다. 전체 미팅 과정은 1-2시간($10-20 GPT-4o 비용)에 완료되었다.

Motivation

  • 알려진 것: 학제간 과학 연구는 점점 더 큰 팀과 다양한 전문성을 요구하며(AlphaFold 2 논문 저자 34명), LLM은 과학적 질문 답변에서 인간 과학자를 match하거나 능가하는 능력을 보임
  • Gap: 기존 LLM 연구 도구들(ChemCrow, Coscientist, AI Scientist)은 좁은 단일 도메인에 국한되거나 실세계 실험 검증이 없었음. 개방형 학제간 연구 문제에 LLM을 적용한 사례가 부재
  • 왜 중요한가: 모든 연구자가 여러 분야의 전문가에 접근할 수 있는 것은 아니며, 특히 자원이 부족한 그룹에게 학제간 협업은 심각한 장벽. SARS-CoV-2는 빠르게 기존 항체/나노바디 치료에 내성을 획득하고 있어 신속한 신규 나노바디 개발이 필수적
  • 접근법: LLM 에이전트 팀(PI, Scientific Critic, 분야별 과학자)이 팀 미팅과 개별 미팅을 통해 연구 설계부터 코드 구현까지 수행하고, 인간 연구자가 고수준 방향 제시 및 실험 검증을 담당하는 AI-인간 협업 프레임워크

Achievement

  1. 자기설계 파이프라인: AI 에이전트가 ESM(단백질 언어 모델) + AlphaFold-Multimer(구조 예측) + Rosetta(결합 에너지)를 통합한 반복적 돌연변이 최적화 워크플로우를 자체 설계. 가중 점수 공식 WS = 0.2(ESM LLR) + 0.5(AF ipLDDT) - 0.3*(RS dG)도 에이전트가 결정
  2. 뛰어난 발현 성공률: 92개 돌연변이 나노바디 중 93.5%(86/92)가 >5 mg/L 발현량, 38%(35/92)가 >25 mg/L의 높은 발현량을 시현
  3. 원래 결합력 유지: H11-D4/Nb21 시리즈의 96%(44/46)가 Wuhan RBD 결합력 유지
  4. 최신 변이체 결합 후보 발견: Nb21 돌연변이(I77V-L59E-Q87A-R37Q)가 JN.1 RBD에 결합(EC50 2.0 vs Wuhan 0.2 ng/mL) + KP.3 RBD에 결합 증거(intensity 3.5 vs 다른 돌연변이 평균 0.06). Ty1 돌연변이(V32F-G59D-N54S-F32S)가 Wuhan RBD 결합력 향상 + JN.1 RBD에 중간 수준 결합 획득
  5. 극도의 효율성: 전체 미팅 1-2시간, $10-20 비용. 인간 입력은 전체 텍스트의 1.3%(1,596단어). 파이프라인 설계에 수일(인간 단독 대비 수주), 계산 파이프라인 실행 약 1주, 합성 약 6주, 결합 실험 약 2주

How

  • 에이전트 아키텍처: GPT-4o 기반. PI(AI for research 전문) + Scientific Critic(오류 검증) + 3명의 과학자 에이전트(면역학, 계산생물학, ML). 각 에이전트는 title, expertise, goal, role 4가지 기준으로 정의
  • 5단계 워크플로우:
  • 팀 선발: PI가 개별 미팅(5회 병렬 + merge)에서 과학자 에이전트 자동 생성
  • 프로젝트 사양: 팀 미팅에서 나노바디 vs 항체, 대상 변이체(KP.3) 등 고수준 결정
  • 도구 선택: 팀 미팅에서 ESM, AlphaFold-Multimer, Rosetta를 선정
  • 도구 구현: 개별 미팅에서 각 도구의 코드를 에이전트가 작성 (Scientific Critic 피드백)
  • 워크플로우 설계: PI가 3개 도구의 통합 워크플로우 결정
  • 반복적 최적화: 4개 시작 나노바디에서 4라운드 반복. 각 라운드: ESM LLR → 상위 20개 → AlphaFold-Multimer ipLDDT → Rosetta dG → WS로 상위 5개 선택 → 다음 라운드. 최종 나노바디당 23개(총 92개)를 수정된 가중 점수(WSWT)로 선택
  • 병렬 미팅: 동일 미팅을 높은 temperature(0.8)로 여러 번 병렬 실행 후, 낮은 temperature(0.2)로 merge하여 창의성과 일관성을 동시 확보
  • 실험 검증: E. coli 발현 → 용해성 단백질 periplasm 분리 → ELISA로 Wuhan/BA.2/JN.1/KP.2.3/KP.3 RBD 패널 결합 프로파일링

Originality

  • AI를 도구가 아닌 연구 설계자로: 기존 접근법은 AlphaFold 등을 인간이 선택한 도구로 사용하는 반면, Virtual Lab에서는 LLM 에이전트가 어떤 도구를 어떻게 조합할지 고수준 의사결정을 자체적으로 수행하는 패러다임 전환
  • 에이전트 정체성의 효과: 서로 다른 과학적 배경을 가진 에이전트들이 다각적 관점에서 토론하여 포괄적 답변 도출. Ablation 실험에서 배경 없는 일반 에이전트보다 우수한 결과 확인
  • Scientific Critic 메커니즘: 전담 비평 에이전트가 다른 에이전트의 답변을 지속적으로 검증하여 품질 향상. Chain-of-thought prompting을 다중 에이전트 관점과 human-in-the-loop로 확장한 구조
  • Nature 게재의 이정표: LLM 멀티 에이전트 시스템이 실험 검증까지 완료하여 Nature에 게재된 최초 사례로, AI for Science 분야의 중요한 이정표

Limitation & Further Study

저자들이 언급한 한계

  • LLM 지식 단절(knowledge cutoff): AlphaFold 3 대신 AlphaFold-Multimer를 사용한 것처럼 에이전트가 최신 도구/문헌을 인식하지 못할 수 있음. RAG나 파인튜닝으로 완화 가능
  • 프롬프트 엔지니어링 의존: 적절한 안내 없이는 에이전트가 모호한 답변을 제공하므로 인간 연구자의 반복적 프롬프트 조정이 필요
  • 환각(hallucination) 위험: 에이전트가 부정확한 과학적 사실이나 인용을 만들어낼 수 있어 인간의 핵심 사실 검증이 필수
  • 코드 디버깅 필요: 에이전트가 작성한 코드에 오류가 있어 추가 개별 미팅을 통한 수정이 필요했음

자체판단 아쉬운 점

  • 유망 후보의 낮은 비율: 92개 중 유망 후보가 2개(2.2%)에 불과하며, 해당 후보의 결합력도 moderate 수준(Nb21 돌연변이의 JN.1 EC50이 Wuhan 대비 10배 약함). 치료 목적 개발까지는 추가 최적화가 필수적
  • Ty1 시리즈 실패: Ty1 돌연변이 23개 중 절반 이상이 Wuhan RBD 결합력을 잃었으며, 이는 첫 돌연변이 위치(position 32) 선택의 문제로 추정. 에이전트의 돌연변이 위치 선택 전략에 대한 심층 분석이 부족
  • 단일 응용 분야: 나노바디 설계라는 비교적 정의가 잘 된 문제에만 적용되어, 더 개방적인 연구 질문이나 다른 과학 분야로의 일반화는 미검증
  • 인간 개입의 역할 불투명: 인간 연구자가 1.3%의 텍스트만 기여했다고 하나, 프롬프트 튜닝과 코드 디버깅에 수일이 소요됨. 이 "비가시적" 인간 기여의 실질적 중요도가 과소평가되었을 가능성
  • 비교 기준 부재: 인간 전문가 팀이 동일 문제를 풀었을 때의 결과와 직접 비교하지 않아, Virtual Lab의 실질적 부가가치를 정량적으로 평가하기 어려움

후속 연구

  • 약물 발견, 재료 과학, 합성 생물학 등 다른 학제간 연구 문제로의 일반화 검증
  • 에이전트에 RAG 및 실시간 문헌/도구 접근 기능을 통합하여 knowledge cutoff 문제 해결
  • 샌드박스 환경에서 에이전트가 독립적으로 도구 설치, 코드 작성, 실행, 디버깅까지 수행하는 완전 자율 시스템으로 발전
  • Google의 AI co-scientist(가설 생성/검증 중심)와의 통합 또는 비교 연구를 통한 상호 보완적 발전
  • 유망 후보 나노바디 2개에 대한 추가 최적화 및 in vivo 효능/안전성 검증
