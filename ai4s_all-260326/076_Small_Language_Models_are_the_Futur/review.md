# Small Language Models are the Future of Agentic AI

- **저자**: Peter Belcak, Greg Heinrich, Shizhe Diao, Yonggan Fu, Xin Dong, Saurav Muralidharan, Yingyan Celine Lin, Pavlo Molchanov
- **소속**: NVIDIA Research, Georgia Institute of Technology
- **DOI**: [10.48550/arXiv.2506.02153](https://doi.org/10.48550/arXiv.2506.02153)
- **발행일**: 2025-06-02
- **키워드**: Small Language Models, Agentic AI, LLM-to-SLM Conversion, Inference Efficiency, Heterogeneous Agents

---

## Essence

에이전틱 AI 시스템에서 대형 언어 모델(LLM)을 소형 언어 모델(SLM, ~10B 파라미터 이하)로 대체해야 한다는 포지션 페이퍼이다. SLM이 (V1) 에이전틱 작업에 **충분한 성능**을 보유하고, (V2) 운영적으로 **더 적합**하며, (V3) **더 경제적**이라는 세 가지 핵심 주장을 전개한다. 범용 대화 능력이 필요한 경우에는 SLM을 기본으로 하고 LLM을 선별적으로 호출하는 **이질적(heterogeneous) 에이전틱 시스템**을 제안한다. LLM에서 SLM으로의 전환을 위한 6단계 변환 알고리즘도 제시한다.

---

## Motivation

현재 에이전틱 AI 산업은 거의 전적으로 LLM API 엔드포인트에 의존하고 있으며, 2024년 기준 LLM API 시장 56억 달러 대비 클라우드 인프라 투자가 570억 달러에 달하는 10배 격차가 존재한다. 그러나 에이전틱 시스템 내 대부분의 LM 호출은 반복적이고, 범위가 제한적이며, 비대화적인 특수 작업이다. 이러한 단순 작업에 범용 LLM을 사용하는 것은 계산 자원의 비효율적 배분이며, 경제적으로 비효율적이고 환경적으로도 지속 불가능하다. 업계의 관성과 대규모 선행 투자가 이 비효율을 지속시키고 있다.

---

## Achievement

1. **체계적 포지션 정립**: SLM의 충분성(V1), 적합성(V2), 경제성(V3)에 대한 구조화된 논증 프레임워크 제시
2. **반론 분석 및 반박**: LLM의 일반 언어 이해 우위(AV1), 중앙화 경제성(AV2), 산업 관성(AV3) 등 주요 반론을 체계적으로 검토하고 반박
3. **LLM-to-SLM 변환 알고리즘**: 데이터 수집 -> 큐레이션 -> 태스크 클러스터링 -> SLM 선택 -> 파인튜닝 -> 반복 개선의 6단계 실용 가이드
4. **사례 연구**: MetaGPT(60%), Open Operator(40%), Cradle(70%)에서 SLM 대체 가능 비율 추정

## How

### 논증 구조 (Arguments A1-A7)
- **A1 (충분한 성능)**: Phi-2/3, Nemotron-H, SmolLM2, Hymba-1.5B, DeepSeek-R1-Distill, RETRO-7.5B, xLAM-2-8B 등 최신 SLM이 commonsense reasoning, tool calling, code generation, instruction following에서 10-25배 큰 모델과 동등한 성능 달성
- **A2 (경제성)**: 7B SLM 서빙이 70-175B LLM 대비 10-30배 저렴 (지연시간, 에너지, FLOPs). LoRA/DoRA 파인튜닝은 수 GPU-hours만 소요
- **A3 (유연성)**: 소규모 전문가 모델의 빠른 학습/배포/적응 가능, AI 민주화 촉진
- **A4 (좁은 기능 노출)**: 에이전트는 LM의 극히 일부 기능만 사용 -- 정교한 프롬프트로 제한된 영역에서 동작
- **A5 (행동 정렬)**: 에이전틱 상호작용은 엄격한 포맷 준수 요구 -- 단일 포맷에 특화된 SLM이 범용 LLM보다 안정적
- **A6 (자연적 이질성)**: 에이전틱 시스템은 본질적으로 다중 모델 조합을 허용하는 모듈형 구조
- **A7 (데이터 자동 수집)**: 에이전틱 상호작용 자체가 SLM 파인튜닝용 고품질 데이터의 자연스러운 원천

### 반론 대응
- **AV1 (LLM의 일반 이해 우위)**: 스케일링 법칙은 아키텍처 고정 가정, SLM은 다른 아키텍처(Mamba-Transformer 하이브리드 등)로 효율 극대화 가능. 에이전틱 시스템의 과업 분해로 "semantic hub"의 효용 약화
- **AV2 (중앙화 경제성)**: 인프라 비용 지속 하락, 추론 스케줄링 기술(NVIDIA Dynamo 등) 발전으로 분산 SLM 서빙의 경제성 개선
- **AV3 (산업 관성)**: 실질적 장벽이나 기술적 한계는 아님

### 변환 알고리즘 (S1-S6)
S1 사용 데이터 로깅 -> S2 PII 제거 및 큐레이션 -> S3 비지도 클러스터링으로 태스크 식별 -> S4 후보 SLM 선택 -> S5 LoRA/QLoRA/지식 증류로 특화 파인튜닝 -> S6 반복 개선

## Originality

1. **"SLM-first" 패러다임 제안**: 에이전틱 AI에서 SLM을 기본으로, LLM을 선택적으로 호출하는 아키텍처 원칙을 명시적으로 주장한 최초의 체계적 포지션 페이퍼
2. **가치 진술(value statement) 형식**: 권고나 의무가 아닌, 커뮤니티 가치의 충실한 반영으로서의 필연적 발전 방향을 진술
3. **실용적 변환 경로**: 기존 LLM 기반 에이전트를 SLM으로 점진적으로 전환하는 구체적 알고리즘 제공
4. **개방적 토론 촉구**: 반론을 환영하고 모든 대응을 공개 게시하겠다는 약속 (research.nvidia.com/labs/lpr/slm-agents)

## Limitation & Further Study

### 한계
- **포지션 페이퍼의 본질적 한계**: 실증적 벤치마크 비교 실험이 없음 -- 주장은 기존 문헌의 개별 결과를 종합한 것
- **SLM 대체 비율의 주관성**: 사례 연구(MetaGPT 60%, Open Operator 40%, Cradle 70%)의 추정치가 정량적 실험이 아닌 정성적 판단에 기반
- **복잡한 추론 과업의 경계 불분명**: "SLM으로 충분한 과업"과 "LLM이 필요한 과업"의 경계를 명확히 정의하지 못함
- **NVIDIA 소속 저자의 잠재적 이해 충돌**: SLM 에지 추론 인프라(ChatRTX, Dynamo 등)를 판매하는 NVIDIA의 사업적 이해와 연구 주장의 일치
- **멀티모달 에이전트 미고려**: 비전-언어 모델로의 확장 가능성만 언급, 실질적 분석 부재

### 향후 연구
- SLM vs. LLM의 에이전틱 태스크별 성능 비교 벤치마크 구축
- 이질적 에이전틱 시스템에서의 최적 모델 라우팅 전략 연구
- 변환 알고리즘의 대규모 실증 검증
- 에이전틱 SLM 전용 평가 벤치마크 개발 (범용 벤치마크 대신)
- 온디바이스 SLM 에이전트의 프라이버시/보안 이점 정량화

## Evaluation

### 강점
- 논증 구조가 매우 체계적이며, 주장(V1-V3), 논거(A1-A7), 반론(AV1-AV3), 반박의 4층 구조로 논리적 완결성이 높음
- 업계 현실(인프라 투자, 시장 규모, 산업 관성)을 솔직하게 인정하면서도 근본적 비효율을 지적
- SLM 기술의 최신 발전(Phi, Nemotron-H, Hymba, DeepSeek-R1-Distill 등)을 포괄적으로 정리
- 실용적 변환 알고리즘(S1-S6)이 구체적이고 따라할 수 있는 수준
- 개방적 토론 촉구와 반론 공개 약속이 학술적 정직성을 보여줌

### 약점
- 실증적 비교 실험 부재가 가장 큰 약점 -- "충분하다"는 주장을 개별 모델의 벤치마크 성능으로만 뒷받침
- NVIDIA 제품(ChatRTX, Dynamo, Nemotron-H, Hymba)이 빈번히 언급되어 중립성 의문
- SLM 파인튜닝의 누적 비용(다수의 전문가 모델 관리)이 단일 LLM 엔드포인트 대비 실제로 경제적인지 정량 분석 부재
- 에이전틱 과업의 "복잡도" 분류 기준이 모호하여, 어떤 호출이 SLM/LLM에 적합한지 판단하기 어려움

### 총평
에이전틱 AI의 미래 아키텍처에 대한 중요한 방향 제시를 하는 시의적절한 포지션 페이퍼이다. LLM 중심 패러다임의 비효율을 체계적으로 지적하고, SLM-first 아키텍처로의 전환 논거를 잘 구성했다. 다만 포지션 페이퍼 특성상 실증 데이터가 부족하며, NVIDIA 소속 저자의 사업적 이해와의 정렬이 순수 학술적 관점에서 주의를 요한다. 향후 에이전틱 SLM 벤치마크와 대규모 비교 실험이 이 포지션의 타당성을 검증하는 핵심이 될 것이다.
